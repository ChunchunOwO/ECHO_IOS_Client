# Release 更新日志

## 0.5.0 - 独立播放器、本地曲库、真 DSP 与外源数据

发布日期：2026-06-27

这一版把 ECHO iPhone 明确整理为独立音乐播放器：手机本地曲库、ECHO 曲库、控制模式、串流模式、原生 DSP、外源歌词 / 封面和设置持久化都接入到同一个播放体验里。

### 播放与曲库

- 新增本地音乐模块，支持导入、扫描、读取元数据、读取本地 LRC、删除本地歌曲和统计本地音乐占用空间。
- 曲库页可在 ECHO 曲库和手机本地曲库之间切换。
- 本地曲库支持歌曲、专辑、艺术家、格式、收藏、最近播放等视图。
- 播放页输出模式改为本地、控制、串流三种模式。
- 本地播放支持本地队列、上一首、下一首、播放 / 暂停、单曲循环、进度拖动和音量控制。
- 播放列表弹窗加入开合动画，并支持本地队列操作。
- Now Playing 副标题按当前模式显示为 `Local Mode`、`Streaming Mode` 或 `Controlling Mode`。

### DSP 与音频

- 新增 iOS 原生 DSP 模块 `echo-audio-dsp`，使用 `AVAudioEngine`、`AVAudioPlayerNode`、`AVAudioUnitEQ` 和动态处理器。
- 本地播放和串流播放优先走 DSP 链路，EQ、响度归一化、音量和 seek 会真实作用到音频播放。
- 新增 EQ 预设：均衡、低频、人声、清晰、暖声、夜间。
- 播放页新增 EQ 按钮和 EQ 面板，设置页也可以修改 EQ。
- 新增响度归一化设置，默认关闭。
- 音频 tag 支持选择展示项：输出模式、来源、串流能力、格式 / 音质、码率和时长。
- 支持展示 ECHO 端返回的专业音频字段：`codec`、`sampleRate`、`bitDepth`、`bitrate`。

### 连接与错误提示

- 新增 ECHO 连接总开关，默认关闭；关闭时不会自动连接、轮询或弹出连接异常。
- 连接页标题简化为“连接”。
- 连接页新增“连接 ECHO / 流媒体”切换；流媒体入口暂时显示“正在制作，暂未开放”。
- 手动连接和配对连接继续支持 EchoLink Host、Port、Token。
- EchoLink Host、Port、Token 会保存到本地，不需要每次重新粘贴配对链接。
- 连接、曲库和本地播放错误改为弹窗提示，同类错误只提示一次，减少操作打断。
- 连接状态文案统一，“连接已断开”改为“ECHO未连接”。

### 设置与持久化

- 设置页重构为分组列表，点击列表项后展开对应设置。
- 语言设置接入全局文案，支持中文 / English。
- 新增默认页面、默认曲库、默认本地曲库视图等设置。
- 新增外源数据设置分组，默认关闭。LRCLIB 和网易云音乐可单独开启。
- 设置项保存到 App 本地个人数据，包括语言、默认页面、默认曲库、音频 tag、EQ、响度归一化、ECHO 连接开关、LRCLIB 开关和网易云音乐开关。
- 本地音乐状态会继续保存，包括收藏、最近播放和本地播放队列。

### 外源歌词与封面

- LRCLIB 用于优先获取歌曲歌词。
- 网易云音乐作为中文曲库补充，主要补充封面，也可作为歌词 fallback。
- 外源数据支持本地播放、串流播放和控制模式，并以歌曲名 + 艺术家作为缓存键。
- 歌词优先级为本地 LRC / EchoLink 歌词优先，外源歌词补位；外源之间优先 LRCLIB，其次网易云音乐。
- 封面优先使用本地文件或 EchoLink artwork URL。
- 如果 EchoLink 没有返回封面，或封面图片加载失败，App 会尝试检索网易云音乐封面。
- 新封面加载成功前会保留上一张稳定封面，减少空白和闪动。

### UI 与交互

- 播放页重新收紧排版，封面、歌曲信息、tag、进度、主控按钮、音量、EQ 和播放列表放进同一个播放器视图。
- 底部 dock 和主要按钮改为玻璃质感。
- 页面切换加入滑动和渐变动效。
- 音量展开条加长，并显示当前百分比。
- 歌词页支持自动滚动、高亮当前歌词和点击有时间戳的歌词 seek。
- 曲库 / 连接 / 设置页顶部移除多余的 `ECHO iPhone` 小标题。
- 曲库页标题下面的说明文字已移除，页面更简洁。

### 已知限制

- LRCLIB 和网易云音乐外源数据需要手机能访问外网。
- 网易云音乐接口为非官方公开接口，后续如果要长期公开发布，建议考虑服务端代理或更稳定的数据源。
- 流媒体页签目前只是占位入口，暂未开放。
- Windows 环境可以验证 TypeScript 和 Expo 导出，但原生 Swift 模块仍需在 macOS / Xcode 或 GitHub Actions 中完成最终编译验证。

### 验证

- `npm.cmd run typecheck`
- `git diff --check`
- `npx.cmd expo export --platform ios --output-dir build\export-check`

## 0.4.0 - 音乐播放器重构、设置页与专业音频 Tag

发布日期：2026-06-23

这一版把 ECHO iPhone 从早期控制界面继续推进成更完整的音乐播放器，重点处理播放页拥挤、组件遮挡、专业音频信息展示和基础设置入口。

### 新增

- 新增设置页，并加入底部 dock。
- 设置页支持界面语言选择：中文 / English。
- 设置页支持选择音频 tag 展示项：输出模式、来源、串流能力、格式音质、码率、时长。
- 曲库新增筛选：全部 / 可串流 / 本地。
- 连接页新增 Host、曲库数量、可串流数量概览。

### 音频信息

- 支持展示 ECHO 桌面端发送的专业音频字段：`codec`、`sampleRate`、`bitDepth`、`bitrate`。
- 音频质量 tag 合并为更易读的格式，例如 `FLAC 48kHz/24bit`。
- 兼容 `48000` 和 `48` 两种采样率输入，避免显示异常。

### 播放体验

- 播放页整体重构为深色玻璃音乐播放器风格。
- 音量控制改为辅助控制行里的可展开滑条，减少常驻占位。
- 上一首 / 下一首改成双箭头，降低与页面滑动或返回手势的视觉冲突。
- 歌词页保留大字号、高对比、自动滚动与点击歌词跳转。
- 播放列表弹窗保持在播放页上方展示，并修复暗色背景下标题对比。

### 布局与稳定性

- 收紧封面、主控按钮、tag 与控制区尺寸，降低小屏重叠风险。
- 歌词页或音量展开时允许滚动，避免 dock 遮挡控制区域。
- 连接页 Host 文本增加单行截断，避免撑破布局。
- 保持手机端改动独立，没有修改 PC 端源码。

### 验证

- `npm.cmd run typecheck`
- `git diff --check`
- `npx.cmd expo export --platform ios --output-dir build\export-check`

### 已知限制

- 语言设置目前先覆盖导航、页面标题/描述和设置页文案，播放页内部分功能文案仍保持中文。
- 设置项持久化已在 0.5.0 补上。
