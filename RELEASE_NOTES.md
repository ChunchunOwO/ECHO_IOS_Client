# Release 更新日志

## 0.3.0 - 全局 UI 重构与文档刷新

这一版把 ECHO iPhone 的视觉和文档都重新整理了一遍，让它更像一个完整的音乐播放器伴侣，而不是原型页面。

### 新增

- 播放页、歌词页、dock 栏统一重构为一套玻璃感界面。
- 播放页改为更满、更紧凑的满版布局。
- 歌词页重做为高对比阅读面板，字号更大，当前行自动滚动。
- 预览图改成更偏 ACG / 二次元气质的功能海报。
- 应用图标更新为更干净的深色音符风格。
- README 和英文 README 重新写成更完整的项目主页风格。

### 优化

- 歌曲封面加载更稳定，不再频繁闪回默认占位图。
- 进度条和音量条拖动时锁定页面手势，减少断触和误滑动。
- dock、按钮、歌词区和播放面板统一成更一致的视觉语言。
- 预览图改为直接介绍功能：EchoLink、歌词、曲库、串流、IPA 构建。

### 继续保留

- EchoLink 配对与手动连接。
- 曲库浏览、点歌、播放控制。
- 手机串流、单曲循环、播放列表预览。
- 未签名 IPA 的 GitHub Actions 构建流程。

### 验证

- `npm.cmd run typecheck`

## 0.2.0 - 播放器界面与歌词体验重构

### 新增

- 播放页新增歌词模式。
- 支持 LRC 时间戳解析与点击跳转。
- 底部控制区加入进度和音量控制。
- 支持左右滑动切换页面。

### 优化

- 控件与 dock 增加玻璃质感。
- 关闭歌词后恢复普通播放器样式。

### 验证

- `npm.cmd run typecheck`

## 0.1.0 - EchoLink iPhone 伴侣原型

- EchoLink 配对链接和手动连接。
- 播放页、曲库页、连接页。
- 播放/暂停、上一首、下一首、进度拖动、音量拖动。
- 曲库浏览与播放。
- 手机串流入口。
- 单曲循环、播放列表预览、封面显示和基础 tag。
